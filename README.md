## 🚀 Como Executar o Projeto

### 1️⃣ Clone o repositório

```bash
git clone https://github.com/jmpinho/cha-lua-de-jade.git
```

1️⃣ Acesse a pasta do projeto:
```bash
cd cha-lua-de-jade
```
2️⃣ Instale as dependências
```bash
npm install
```
3️⃣ Inicie o servidor de desenvolvimento
```bash
cd (nome da pasta)
npx json-server --watch db.json --port 3001
```
4️⃣Execute o programa
```bash
npm run dev
```
O terminal mostrará algo semelhante a:
```bash
http://localhost:5173
```
Abra esse link no navegador para visualizar o projeto
